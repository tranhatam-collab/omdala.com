## 1. GLOBAL STRUCTURE

App = Operator Workspace

Core Navigation:

- Dashboard
- Nodes
- Resources
- Commitments
- Transitions
- Proofs
- Trust
- Messages
- Settings

Purpose:
Control center for current state and suggested actions.

Components:

- Current State Summary
- Desired State Goals
- Active Commitments
- Suggested Transitions (AI)
- Trust Snapshot
