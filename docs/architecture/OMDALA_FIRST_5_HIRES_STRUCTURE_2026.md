Build core system trong 6 tháng đầu.

→ Không tuyển nhiều
→ Tuyển đúng người lõi

## 2. 5 VỊ TRÍ QUAN TRỌNG NHẤT

### 1. Chief Reality Architect

Vai trò:

- Thiết kế toàn bộ system logic
- Khóa primitive model
- Đảm bảo consistency

- Systems thinking cực mạnh
- Product + infra hiểu sâu

---

### 2. Backend Lead (State + Transition)

Vai trò:

- Build core engine
- API v2 /reality
- Event system

Skill:

- Distributed systems
- API design
- Workflow engine

---

### 3. AI / Planner Engineer

Vai trò:

- Transition planning
- Recommendation system
- AI explainability

Skill:

- LLM + planning
- Graph reasoning

---

### 4. Frontend Lead (Reality Map)

Vai trò:

- Build Reality Map UI
- Interaction system

Skill:

- React / WebGL / Graph UI
- UX systems

---

### 5. Trust & Governance Engineer

Vai trò:

- Proof system
- Trust scoring
- Policy engine

Skill:

- Security mindset
- Risk modeling
