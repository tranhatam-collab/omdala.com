# Security Policy

## Reporting

Please report security issues privately to the maintainers. Do not open public issues for exploitable vulnerabilities.

## Scope

This project includes backend API, web orchestration UI, gateway runtime, and iOS app scaffolding.

## Baseline Controls

- policy checks before sensitive execution
- proof logging for physical actions
- role-aware approvals
- transport over TLS in deployed environments
- no sensitive production secrets in repo

## Supported Versions

Security fixes apply to active release-candidate and later branches.
