# Om AI Release Criteria

Version: 1.0
Status: Draft for DEV handoff

## Release Must Have

- sign-in works
- at least one home context works
- supported device control works
- voice and text paths work
- proof is generated
- approvals work
- memory updates work
- failure states are safe

## Release Must Not Have

- silent sensitive execution
- unbounded unsupported device claims
- missing proof for physical actions
- no fallback when speech fails
