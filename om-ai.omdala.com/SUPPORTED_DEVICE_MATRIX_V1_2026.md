# Supported Device Matrix v1 2026

## Purpose

Define the first supported device classes for Om AI.

## Classes

### Tier 1

- Matter native devices
- Apple Home compatible devices

### Tier 2

- Bluetooth accessories with native onboarding
- selected Wi-Fi accessories with supported onboarding

### Tier 3

- local IP devices through Om Gateway
- selected legacy hubs through Om Gateway

### Tier 4

- cloud-connected devices through vendor connector

### Tier 5

- unsupported or unknown devices

## Required Metadata

- supported class
- onboarding path
- execution path
- proof path
- approval path
- fallback path
